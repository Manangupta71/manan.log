# manan.log

A personal blog — notes on machine learning, research, engineering, literature, and everything in between. No tracking, no ads, no algorithm.

**Live site:** [manangupta71.github.io/manan.log](https://manangupta71.github.io/manan.log/)


## Stack

Plain HTML/CSS, no framework, no build step. Hosted on GitHub Pages, deployed from `main`.

## Running locally

Just open `index.html` in a browser — no server needed.
